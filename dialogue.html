Tag guide:

[FS=x]...[/FS]: Changes the font size. The x specifies the font size.

[SW]...[/SW]: Applies a wave effect.

[ZOOM=x-y-z]...[/ZOOM]: Applies a zoom effect. The x is the starting zoom level, y is the ending zoom level, and z is the speed of the zoom.

[PS=x]: Introduces a pause. The x specifies the duration of the pause.

[SPD=x]...[/SPD]: Changes the speed animation. The x specifies the speed.

[SHAKE]...[/SHAKE]: Applies a shake effect.

Font in px | Delay in miliseconds | Tags are not combinable.

Default dialogue speed is 25.
